# printd-vue-component-example

> Example of printing a [Vue 2](https://vuejs.org/) component using [Printd](https://github.com/joseluisq/printd/).

:tada: View demo on [CodePen](https://codepen.io/joseluisq/full/VzRpGb/).

## Install

```sh
yarn
```

## Usage

```sh
yarn start
```
